# Health Conditions Translator Project

This repository serves as a linguistic health tool designed to translate common medical and health conditions between English and Farsi. It provides the terminology, pronunciation guides, and contextual sentences to help individuals, healthcare workers, and language learners navigate health-related communication.

## Contents
- **`health_conditions_1_50.md`**: A comprehensive Markdown document containing the first 50 common health conditions, complete with:
  - English terms
  - Farsi translation
  - Farsi transliteration/pronunciation
  - Example sentences in both languages

## Usage
You can convert the `.md` documents in this repository into PDFs, DOCX, or read them directly on GitHub. To generate PDFs or DOCX files, you can use standard markdown converter tools like Pandoc.
